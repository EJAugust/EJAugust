This is a quick demo showing how the super keyword can be used in any class method in order to gain access to prototype values.

First, we'll let A, B, C be undefined.

Next, we create three simple classes, each with a property called property and each one extending the last:

{
 A = class {
  static get property() {
   return 3;
  }
 };
}
{
 const enclosedValue = 7;
 B = class extends A {
  static get property() {
   return super.property + enclosedValue;
  }
 };
}
{
 const enclosedValue = 9;
 C = class extends B {
  static get property() {
   return super.property + enclosedValue;
  }
 };
}

Results

If super.property works, then we should see the values of all three property properties go up for each successive class.

To test it, we'll simply grab the output of property for all three:

A.property === 3
B.property === 10 // (3 + 7)
C.property === 19 // (3 + 7 + 9)
